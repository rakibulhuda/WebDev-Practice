CSS:
There are three (3) types of :
1. inline css -- > it is written inside the tag
2. embedded css --> we write css inside one html. it is only for that particular page. the css style is not accessed outside that file
3. external css --> once it is written, it is accessed through the entire file.

There are several selector in css:
1. class --> .login{} (written convention)
2. id --> #highlight{}
3. element --> li{}
4. universal --> *{}
5. grouping & attribute
